Присвоение тегов:
сделать новый коммиты
определить sha коммита (выводит последние 5 коммитов): git log --pretty=format:"%h - %an - %ad - %s" --date=format:'%H:%M:%S %d.%m.%y' -n 5
использовать команду: git tag -a v0.0.6 4ee6d06 -m "сообщение в теге"
